QuantCoderFS

A full-stack application that automates trading algorithm generation from academic papers

TypeScript FastAPI CrewAI

QuantCoderFS automates the transformation of academic finance papers into executable trading algorithms. Built with a full-stack architecture using Next.js frontend, FastAPI backend, and CrewAI for multi-agent workflows. Supports both cloud LLMs (OpenAI) and local models via Ollama for privacy-first research automation.

Key Features:

  • PDF paper extraction and summarization
  • Multi-agent pipeline for code generation
  • QuantConnect LEAN integration
  • Local-first AI with Ollama support
  • JWT authentication and user management
