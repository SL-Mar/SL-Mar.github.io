QuantCoderFS v2.0 Documentation

Unified quantitative research and algorithm development platform combining market intelligence with automated strategy generation

Project Status

QuantCoderFS is currently in active development (v2.0). This unified platform merges market intelligence capabilities with automated algorithm generation. This is a research and educational tool - not financial advice.

What is QuantCoderFS?

QuantCoderFS is a unified quantitative research and algorithm development platform that merges two powerful workflows into one seamless experience. It combines AI-powered market intelligence (inspired by the MarketSense AI research paper) with automated strategy generation from academic papers, enabling complete end-to-end quantitative research and deployment.

Market Intelligence Features

  • Multi-Agent AI System - MarketSense AI framework with specialized agents analyzing fundamentals, news sentiment, price dynamics, and macroeconomic conditions
  • Stock Data Aggregation - Comprehensive EOD pricing, fundamentals, and news data
  • Signal Generation - BUY/HOLD/SELL signals with conviction scores and confidence levels
  • TradingView Charts - Interactive price charts with technical indicators
  • Research Notes - Built-in note-taking for trading ideas and observations

Strategy Generation Features

  • PDF Paper Upload - Extract and summarize academic finance papers
  • Automated Code Generation - CrewAI multi-agent workflow generates production-ready QuantConnect algorithms
  • Monaco Code Editor - Professional code editing with syntax highlighting
  • QuantConnect Integration - Push generated code directly to QuantConnect for backtesting and live trading
  • Trading Notes - Generate structured trading notes from strategies

Technology Stack

  • Backend - Python, FastAPI, CrewAI, SQLAlchemy
  • Frontend - Next.js 15, React 19, TypeScript, Tailwind CSS
  • AI Frameworks - CrewAI for agent orchestration, MarketSense AI for signal generation
  • Integrations - QuantConnect REST API, market data providers
  • Database - SQLite (dev), PostgreSQL-ready

MarketSense AI Architecture

QuantCoderFS implements a multi-agent LLM framework based on the MarketSense AI research paper (Fatouros et al., 2024). The system uses five specialized agents working in orchestration:

Research Performance Metrics

In academic testing on S&P 100 stocks over 15 months:

  • Excess alpha: 10-30%
  • Cumulative return: Up to 72%
  • Multi-agent consensus outperformed individual agents significantly

Agent Architecture

  1. Fundamentals Summarizer (30% weight) - Analyzes quarterly financial statements with automated scoring
  2. Progressive News Summarizer (25% weight) - Tracks 60-day rolling news sentiment and narratives
  3. Stock-Price Dynamics Summarizer (25% weight) - Technical analysis with S&P 500 peer comparison
  4. Macroeconomic Environment Summarizer (20% weight) - Two-stage macro analysis (individual → consensus)
  5. Signal-Generation Agent - Master orchestrator with Chain-of-Thought reasoning, produces final BUY/HOLD/SELL recommendation

Quick Start

Get QuantCoderFS running in minutes:

# Clone the repository
git clone https://github.com/SL-Mar/quantcoderfs.git
cd quantcoderfs

# Backend setup
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Frontend setup
cd ../electron
npm install

# Launch the application
cd ..
./launch.sh

For detailed installation instructions, see the Installation Guide.

API Examples

Market Intelligence - Generate Trading Signal

# Generate AI-powered signal for a stock
curl -X POST "http://localhost:8000/api/signals/generate/AAPL.US"

Response includes signal (BUY/HOLD/SELL), conviction score, confidence level, and detailed agent outputs from MarketSense AI framework.

Strategy Generation - Generate Algorithm from Paper

# Upload paper
curl -X POST "http://localhost:8000/api/papers/upload" \
  -F "file=@research_paper.pdf"

# Generate QuantConnect algorithm
curl -X POST "http://localhost:8000/api/papers/generate-code/research_paper.pdf"

Returns generated QuantConnect algorithm code ready for backtesting and deployment.

QuantConnect Integration - Push to Trading Platform

# Push algorithm to QuantConnect
curl -X POST "http://localhost:8000/api/backtest/trigger-backtest" \
  -H "Content-Type: application/json" \
  -d '{"code": "[algorithm code]", "project_name": "My Strategy"}'

Creates QuantConnect project, uploads code, compiles, and starts backtest automatically.

For complete API documentation, see the API Reference.
