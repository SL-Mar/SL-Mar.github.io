EODHD API Endpoints for Biotech Screener

Endpoint 1: Stock Screener API

URL: https://eodhistoricaldata.com/api/screener Purpose: Filter stocks by market cap, sector, volume

bash
GET https://eodhistoricaldata.com/api/screener?api_token=YOUR_API_TOKEN&filters=[
  ["market_capitalisation",">",50000000],
  ["market_capitalisation","<",500000000],
  ["sector","=","Healthcare"],
  ["average_volume",">",100000],
  ["exchange","in",["US","NASDAQ"]]
]&limit=500

Response Example:

  "close": 4.29,

  "volume": 1200000,
  "average_volume": 100000

Endpoint 2: Intraday Data API

URL: https://eodhistoricaldata.com/api/intraday/{TICKER}.US Purpose: Get real-time/pre-market volume and price

bash
GET https://eodhistoricaldata.com/api/intraday/XBIO.US?api_token=YOUR_API_TOKEN&interval=5m&from=1696752000&to=1696838400

Use Case: Detect pre-market volume spikes before market open

Endpoint 3: Historical Data API

URL: https://eodhistoricaldata.com/api/eod/{TICKER}.US Purpose: Calculate 50-day average volume, RSI, moving averages

bash
GET https://eodhistoricaldata.com/api/eod/XBIO.US?api_token=YOUR_API_TOKEN&period=d&from=2025-08-20&to=2025-10-08

Response: Array of daily OHLCV data for technical calculations

Endpoint 4: Technical Indicators API

URL: https://eodhistoricaldata.com/api/technical/{TICKER}.US Purpose: Get pre-calculated RSI, SMA, EMA

bash
GET https://eodhistoricaldata.com/api/technical/XBIO.US?api_token=YOUR_API_TOKEN&function=rsi&period=14
GET https://eodhistoricaldata.com/api/technical/XBIO.US?api_token=YOUR_API_TOKEN&function=sma&period=20
GET https://eodhistoricaldata.com/api/technical/XBIO.US?api_token=YOUR_API_TOKEN&function=sma&period=50

Benefit: No need to calculate RSI/MA manually in n8n Code nodes

    #### Node 7: Technical Confirmation Filter 

    javascript
// Code Node: Apply Technical Filters
const rsiData = $json; // Latest RSI value
const rsi = rsiData[rsiData.length - 1].rsi;
    const historicalData = $('Get 50-Day History').item.json;
const recentPrices = historicalData.slice(-50).map(d => d.close);
    // Calculate 20-day and 50-day MA
const ma20 = recentPrices.slice(-20).reduce((a,b) => a+b, 0) / 20;
const ma50 = recentPrices.reduce((a,b) => a+b, 0) / 50;
    const currentPrice = $('EODHD Universe Filter').item.json.close;
const volumeBreakout = $('EODHD Universe Filter').item.json.volumeBreakout;
    // Apply filters
const passesFilter = (
  volumeBreakout === true &&
  rsi >= 60 && rsi <= 75 &&
  currentPrice > ma20 &&
  currentPrice > ma50
);
    if (!passesFilter) {
  return []; // Filter out
}
    return {
  json: {
    ...($('EODHD Universe Filter').item.json),
    rsi: rsi,
    ma20: ma20,
    ma50: ma50,
    priceAboveMA20: ((currentPrice - ma20) / ma20 * 100).toFixed(2),
    priceAboveMA50: ((currentPrice - ma50) / ma50 * 100).toFixed(2)
  }
};

    #### Node 8: Calculate Composite Score 

    javascript
// Code Node: Scoring Algorithm
const data = $json;
    let score = 0;
    // Volume component (max 30 points)
score += Math.min(data.volumeRatio / 10 * 30, 30);
    // RSI component (max 25 points)
if (data.rsi >= 65 && data.rsi <= 70) score += 25;
else if (data.rsi >= 60 && data.rsi < 65) score += 20;
else if (data.rsi > 70 && data.rsi <= 75) score += 15;
    // Price action component (max 25 points)
const aboveMA20Pct = parseFloat(data.priceAboveMA20);
const aboveMA50Pct = parseFloat(data.priceAboveMA50);
score += (aboveMA20Pct > 5) ? 15 : 10;
score += (aboveMA50Pct > 10) ? 10 : 5;
    // Market cap component (max 20 points)
if (data.market_capitalisation >= 50000000 && data.market_capitalisation <= 150000000) {
  score += 20;
} else if (data.market_capitalisation > 150000000 && data.market_capitalisation <= 300000000) {
  score += 15;
} else {
  score += 10;
}
    return {
  json: {
    ...data,
    compositeScore: Math.round(score)
  }
};
