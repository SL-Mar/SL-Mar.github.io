QuantCoderFS

A full-stack application that automates trading algorithm generation from academic papers

TypeScript FastAPI GLM-4.6 Last updated: 2025-01-11

QuantCoderFS automates the transformation of academic finance papers into executable trading algorithms. Built with a full-stack architecture using Next.js frontend, FastAPI backend, and multi-agent workflows. Currently under refactoring to move to local AI using GLM-4.6 LLM for enhanced privacy, reduced costs, and improved performance in quantitative research automation.

Key Features:

  • PDF paper extraction and summarization
  • Multi-agent pipeline for code generation
  • QuantConnect LEAN integration
  • Local-first AI with GLM-4.6 LLM
  • JWT authentication and user management
