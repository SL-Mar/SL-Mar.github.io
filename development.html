QuantCoderFS

Desktop application for algorithmic trading research and development

Electron TypeScript QuantConnect Last updated: 2025-01-11

QuantCoderFS is a comprehensive desktop application for algorithmic trading research and development. Built with Electron for cross-platform compatibility, it features an integrated PDF viewer for research papers, strategy backtesting, live algorithm deployment to QuantConnect, and multi-workspace management for organizing multiple trading strategies. Streamlines the workflow from research to deployment with integrated tools for quantitative finance.

Key Features:

  • Integrated PDF viewer for research papers
  • Strategy backtesting with QuantConnect LEAN
  • Live algorithm deployment to QuantConnect
  • Multi-workspace management for strategy organization
  • Real-time backtest monitoring and results visualization
  • Cross-platform desktop application (Windows, macOS, Linux)

Chat with SOLAS

RAG-powered knowledge management system for maritime regulations

TypeScript FastAPI Qdrant Last updated: October 2025

AI-powered document management and query system for maritime conventions including SOLAS, MARPOL, STCW, and other IMO regulations. Uses RAG (Retrieval-Augmented Generation) with vector embeddings to provide accurate, citation-backed answers to regulatory questions. Built with FastAPI backend, React frontend, and Qdrant vector database.

Key Features:

  • PDF document indexing with smart chunking (~800 chars)
  • Semantic search with OpenAI embeddings (text-embedding-3-small)
  • Inline PDF viewer with page navigation and zoom controls
  • Chat interface with citation tracking (chapter, section, page)
  • Support for 48+ maritime convention PDFs (SOLAS, MARPOL, etc.)
  • Background indexing with Telegram notifications

Agentic SMS

Workflow automation framework for marine safety management systems

n8n Python QHSE Last updated: 2025

Designing the n8n of Marine Safety Management Systems. Workflow automation framework for ship operations with AI-powered decision support for maritime safety, quality, health, and environmental management. Enables SMS-based natural language interactions via Twilio, powered by multi-agent AI workflows.

Key Features:

  • SMS-based natural language queries for ship operations
  • Multi-agent workflows with CrewAI for task orchestration
  • Web search and maritime information retrieval
  • Automated task management and notifications
  • Context-aware conversations for operational support
  • Integration with n8n for workflow automation
