AlphaDesk Documentation

AI-powered financial analysis platform for quantitative traders and market researchers

Project Status

AlphaDesk is currently in active development (v0.2.0). Features and architecture are subject to change. This is a research and educational project - not financial advice.

What is AlphaDesk?

AlphaDesk is a comprehensive financial data aggregation and analysis platform that combines institutional-grade market intelligence with cutting-edge AI technology. Inspired by the MarketSense AI research paper, AlphaDesk implements a multi-agent framework for sophisticated market analysis.

Key Features

  • Multi-Agent AI System - Specialized agents analyze fundamentals, news sentiment, price dynamics, and macroeconomic conditions
  • Real-Time Market Data - Comprehensive EOD pricing, fundamentals, and news from EODHD API
  • Signal Generation - BUY/HOLD/SELL signals with conviction scores and confidence levels
  • LLM Provider Support - OpenAI, Anthropic Claude, and local models via Ollama
  • Modern UI - Next.js frontend with GitHub dark theme and TradingView charts
  • REST API - Clean FastAPI backend with comprehensive endpoints

Research Performance Metrics

In academic testing on S&P 100 stocks over 15 months:

  • Excess alpha: 10-30%
  • Cumulative return: Up to 72%
  • Multi-agent consensus outperformed individual agents significantly

Agent Architecture

  1. Fundamentals Summarizer (30% weight) - Analyzes quarterly financial statements with automated scoring
  2. Progressive News Summarizer (25% weight) - Tracks 60-day rolling news sentiment and narratives
  3. Stock-Price Dynamics Summarizer (25% weight) - Technical analysis with S&P 500 peer comparison
  4. Macroeconomic Environment Summarizer (20% weight) - Two-stage macro analysis (individual → consensus)
  5. Signal-Generation Agent - Master orchestrator with Chain-of-Thought reasoning, produces final BUY/HOLD/SELL recommendation

Quick Start

Get AlphaDesk running in minutes:

# Clone the repository
git clone https://github.com/SL-Mar/alphadesk.git
cd alphadesk

# Backend setup
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
alembic upgrade head

# Frontend setup
cd ../electron
npm install

# Launch the application
cd ..
./launch-ad.sh

For detailed installation instructions, see the Installation Guide.

API Example

Generate a trading signal for a stock:

# Using OpenAI (default)
curl -X POST "http://localhost:8004/api/signals/generate/AAPL.US"

# Using Ollama (local LLM)
curl -X POST "http://localhost:8004/api/signals/generate/AAPL.US?llm_provider=ollama"

# Using Anthropic Claude
curl -X POST "http://localhost:8004/api/signals/generate/AAPL.US?llm_provider=anthropic"

Response includes signal (BUY/HOLD/SELL), conviction score, confidence level, and detailed agent outputs. See the Signals API documentation for complete details.
