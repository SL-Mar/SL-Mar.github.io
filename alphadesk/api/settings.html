Settings API

Manage API keys and application configuration

Endpoints

GET /api/settings

Retrieve current settings (API keys are masked).

PUT /api/settings

Update settings including API keys.

Request Body

{
  "eodhd_api_key": "your_eodhd_key",
  "openai_api_key": "your_openai_key",
  "anthropic_api_key": "your_anthropic_key",
  "tavily_api_key": "your_tavily_key"
}

Configure these settings through the UI at http://localhost:3004/settings or via API.
