Installation Guide

Complete setup instructions for AlphaDesk

Prerequisites

  • Python 3.12 or higher
  • Node.js 18 or higher
  • EODHD API Key - Get one here (free tier available)
  • Git for cloning the repository

Step 1: Clone the Repository

git clone https://github.com/SL-Mar/alphadesk.git
cd alphadesk

Step 2: Backend Setup

Create Virtual Environment

cd backend
python3 -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

Install Dependencies

pip install -r requirements.txt

Initialize Database

alembic upgrade head
Database Migrations

AlphaDesk uses Alembic for database migrations. The command above creates the SQLite database with all necessary tables.

Step 3: Frontend Setup

cd ../electron
npm install

Step 4: Configuration

Create Environment File

echo "NEXT_PUBLIC_API_BASE_URL=http://localhost:8004" > electron/.env.local

Step 5: Launch Application

Using Launch Script (Recommended)

cd ..
chmod +x launch-ad.sh
./launch-ad.sh

Manual Launch

# Terminal 1 - Backend
cd backend
source .venv/bin/activate
uvicorn main:app --reload --port 8004

# Terminal 2 - Frontend
cd electron
npm run dev -- -p 3004

Step 6: Configure API Keys

  1. Open your browser to http://localhost:3004
  2. Navigate to Settings page
  3. Enter your EODHD API key
  4. (Optional) Enter OpenAI API key for signal generation
  5. (Optional) Configure Anthropic or Ollama for local LLMs
Installation Complete!

AlphaDesk should now be running at:

  • Frontend: http://localhost:3004
  • Backend API: http://localhost:8004
  • API Docs: http://localhost:8004/docs

Verification

Test your installation:

1. Check Backend Health

curl http://localhost:8004/
# Should return: {"message": "AlphaDesk up and running"}

2. Create Your First Stock Project

curl -X POST "http://localhost:8004/api/stocks" \
  -H "Content-Type: application/json" \
  -d '{"ticker": "AAPL.US"}'

3. Access Frontend

Open http://localhost:3004 in your browser and you should see the AlphaDesk dashboard.

Troubleshooting

Port Already in Use

If ports 3004 or 8004 are already in use, you can change them:

# Backend - different port
uvicorn main:app --reload --port 8005

# Frontend - different port
npm run dev -- -p 3005

# Update .env.local accordingly
echo "NEXT_PUBLIC_API_BASE_URL=http://localhost:8005" > electron/.env.local

Python Version Issues

Ensure you're using Python 3.12+:

python3 --version

Node.js Version Issues

Check your Node.js version:

node --version  # Should be v18 or higher

Database Errors

If you encounter database errors, try resetting:

cd backend
rm alphadesk.db  # Remove existing database
alembic upgrade head  # Recreate
