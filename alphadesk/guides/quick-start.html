Quick Start Guide

Get your first analysis running in 5 minutes

This guide assumes you've completed the installation.

Step 1: Start AlphaDesk

./launch-ad.sh

Step 2: Create a Stock Project

curl -X POST "http://localhost:8004/api/stocks" \
  -H "Content-Type: application/json" \
  -d '{"ticker": "AAPL.US"}'

Step 3: Generate a Trading Signal

curl -X POST "http://localhost:8004/api/signals/generate/AAPL.US"

The response will include BUY/HOLD/SELL recommendation with AI analysis. See Signals API for details.
